# 📏 PixelRuler

**PixelRuler** is a lightweight, accurate, and easy-to-use desktop utility designed for developers and UI/UX designers to measure screen dimensions in pixels. Built with Java, it provides a simple overlay to measure any object on your screen in real-time.

---

## ✨ Features

*   **Real-time Tracking:** Accurate pixel measurement with a dynamic vertical cursor.
*   **Dual Orientation:** Switch between **Horizontal** and **Vertical** modes via the right-click menu.
*   **Always on Top:** Stays above all other windows for seamless measuring.
*   **Draggable Interface:** Click and drag to move the ruler anywhere on your screen.
*   **Native Look:** Features a clean and professional UI (supports FlatLaf).
*   **Lightweight:** Extremely low resource consumption.

---

## 🚀 Installation & Usage

### For Users (Executable)
1.  Go to the [Releases](https://github.com/Ramika-N/PixelRuler/releases) section.
2.  Download the latest `PixelRuler.exe`.
3.  Run the application. 
    *   *Note: If you see a "Windows protected your PC" message, click **More Info** -> **Run anyway**. This is a safe application created by an independent developer.*

### For Developers (Source Code)
1.  Clone the repository:
    ```bash
    git clone [https://github.com/Ramika-N/PixelRuler.git](https://github.com/Ramika-N/PixelRuler.git)
    ```
2.  Open the project in **NetBeans IDE**.
3.  Ensure you have **Java 17 or higher** installed.
4.  Build and Run!
